# Dossier Public - Assets Statiques

## Fichiers à placer ici:

### 1. logo.png
- **Dimensions recommandées**: 200x200px ou 400x400px
- **Format**: PNG avec transparence
- **Usage**: Logo affiché dans le header de l'application
- Doit être visuellement clair et professionnel

### 2. favicon.png
- **Dimensions recommandées**: 32x32px ou 64x64px
- **Format**: PNG
- **Usage**: Icône affichée dans l'onglet du navigateur
- Version simplifiée du logo, lisible en petite taille

## Instructions:
1. Créez ou préparez vos fichiers logo et favicon
2. Nommez-les exactement `logo.png` et `favicon.png`
3. Placez-les dans ce dossier `/public`
4. Commitez et pushez les changements

Si vous n'avez pas encore de logo/favicon, vous pouvez:
- Les créer avec des outils comme Canva, Figma, ou Adobe Illustrator
- Utiliser des générateurs en ligne
- Les créer temporairement pour tester l'application

L'application continuera de fonctionner sans ces fichiers, mais ils sont recommandés pour une présentation professionnelle.
